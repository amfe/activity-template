# activity template

## Install

`adam add tpl-activity`

## Initial

`adam init`

## Usage

+ compile: `npm run compile`
    + compile js: `npm run js`
    + compile css: `npm run css`
+ watch: `npm run watch`
    + watch js: `npm run watch:js`
    + watch css: `npm run watch:css`
+ serve: `npm run serve`
+ dev: `npm run dev`
+ build: `npm run build`
